The supported iOS is changing (it’s not very surprising). More recent released applications supports more recent iOS versions. It’s quite normal. Apple “controls” the supported version of iOS by the configuration of the minimal target version of Xcode.

  • Xcode 3 : Supports minimum target version iOS3.0
  • Xcode 4 : Supports minimum target version iOS4.3
  • Xcode 5 : Supports minimum target version iOS4.3
  • Xcode 6 : Supports minimum target version iOS4.3

Actually, it’s possible to do a iOS application with iOS4.3 compatibility but it’s complicated for developpers.
